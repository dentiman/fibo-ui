Usage

Basic Usage

<button class="btn">Default Button</button>
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-chip">Chip Button</button>
<button class="btn btn-text">Text Button</button>

With Icons

<button class="btn btn-primary">
  <svg class="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M5 13l4 4L19 7"></path>
  </svg>
  Save Changes
</button>

Disabled State

<button class="btn" disabled>Disabled Button</button>
<button class="btn btn-primary" disabled>Disabled Primary</button>

Custom Sizes

<!-- Small -->
<button class="btn text-xs px-2 py-1 h-7">Small</button>

<!-- Large -->
<button class="btn text-base px-6 py-3 h-11">Large</button>

Features:

  • • Four button variants: default, primary, chip, text
  • • Automatic icon padding reduction
  • • Hover and focus states
  • • Disabled state support
  • • Dark mode support
  • • Customizable sizes
  • • Accessible keyboard navigation
  • • Consistent spacing and typography
