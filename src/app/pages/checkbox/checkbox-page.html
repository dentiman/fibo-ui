Usage

Basic Usage with Inline Label

<fibo-checkbox [(checked)]="isChecked">
  Accept terms
</fibo-checkbox>

With Form Control

<fibo-checkbox [formControl]="checkboxControl">
  I agree to the terms
</fibo-checkbox>

With Complex Content

<fibo-checkbox [(checked)]="value" class="items-start">
  <div>
    <div class="font-medium">Title</div>
    <p class="text-xs text-slate-500">Description</p>
  </div>
</fibo-checkbox>

With Loading State

<fibo-checkbox
  [(checked)]="value"
  [isLoading]="loading"
  [disabled]="disabled">
  Loading checkbox
</fibo-checkbox>

With Indeterminate State

<fibo-checkbox
  [(checked)]="value"
  [indeterminate]="true">
  Indeterminate checkbox
</fibo-checkbox>

<fibo-checkbox
  [(checked)]="value"
  [indeterminate]="isIndeterminate">
  Dynamic indeterminate
</fibo-checkbox>

Features:

  • • Inline label support with content projection
  • • Two-way data binding with [(checked)]
  • • Angular Forms integration (FormControl, ngModel)
  • • Keyboard navigation (Space, Enter)
  • • Loading state indicator
  • • Disabled state support
  • • Indeterminate state support
  • • Full accessibility (ARIA attributes)
  • • Custom SVG checkmark icon
  • • Hover and focus states
  • • Support for complex content (titles, descriptions)
