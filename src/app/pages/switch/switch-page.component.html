Usage

Basic Usage

<sui-switch [(checked)]="isEnabled"></sui-switch>

With Form Control

<sui-switch [formControl]="switchControl"></sui-switch>

With Loading State

<sui-switch
  [(checked)]="value"
  [isLoading]="loading"
  [disabled]="disabled">
</sui-switch>

Features:

  • • Two-way data binding with [(checked)]
  • • Angular Forms integration (FormControl, ngModel)
  • • Keyboard navigation (Space, Enter)
  • • Loading state indicator
  • • Disabled state support
  • • Full accessibility (ARIA attributes)
  • • Customizable styling
